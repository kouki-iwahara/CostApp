## 職務経歴 要約

### 2017/2 - 2019/8 :飲食業界(ベンチャー企業)
 職務: 店長
#### 職務内容
- 新規店舗の立ち上げ
- 予算、スタッフマネージメント（月一のミーティング、個人面談）
- 季節ごとの新メーニュー考案、作成
- 幹部クラス対象の経営に関するセミナーに参加（[株式会社武蔵野](https://www.musashino.co.jp/)　[株式会社繁盛塾](https://hanjyoujyuku.co.jp/)）
- 介護施設を対象にした出店ボランティア活動([らぁ祭](http://rasai.jp/))

店長として多くの業務に携わり、また改善も試みました。
最も効果が大きかった取り組みが「スタッフ全員、一日一改善の習慣」です。
「全ての答えは現場にある」をスローガンとし、社長、幹部クラスが各店舗へ偵察。
その気づきを元に、一日一つで良いから改善していこうという運動です。
スタッフが一日４人いれば、一ヶ月で120個改善されます。
日々の改善は小さいですが、一か月前と後では全く別の店に生まれ変わります。

この取り組みの本当の目的は、「みんなで決めた事をみんなでやり遂げる」という強いチームビルディングです。
確かなチームの土壌を作る強い要因になったと思います。

### 2012/12 - 2017/1 :個人事業主として飲食店を経営
 職務: 事業主
#### 職務内容
- 月一回のメニュー切り替え
- お客様からいただいたアドバイスを取り入れたお店作り
- 予算、スタッフマネージメント

小さいですが個人経営でフードバーを経営していました。
翌月の給料が保証されず、「全て自由で全て自己責任」という状況に身を置いて仕事をした事は、非常に良い経験をしたと思っています。

#### 経営するにあたり大事にしていた理念
「お客様と真摯に向き合い、真っ当な仕事をし、真っ当な利益を得る」

これは、我々はお客様に対し一切妥協する事なく仕事をする事（お客様からすれば当然の事）、不当な安売りはせず、それだけに見合う対価を得る事（お店からすれば当然の事）という意味です。

飲食での例ですが、業務が忙しかったりすると、手が汚れてきます。
しかし、忙しいからと言ってその汚い手で作る料理は、お客様にとって真っ当な仕事にはなりません。
できる限りの真っ当な仕事をして、気持ちよく真っ当な利益をいただけるように掲げた理念です。

#### 失敗から学んだ教訓
全ての業務に対して責任があるという経験は、人としても大きく成長できたと思います。
ですが失敗も重ねてきました。
失敗から学んだ一番強く残る教訓は、

「現状維持をしようとすると衰退する」

という事です。

僕はお店のメニューを月一回変えたり、BGMのジャンルや小物の配置など試行錯誤しながら取り組んでいました。

しかし3年ほどたった辺りから、だんだん環境に慣れてきたせいか少しずつメニューを変える頻度が落ち、店内の雰囲気も変わらずの状態になっていきました。でも経営理念の通り決して接客のレベルや料理の味は決して落としてはいません。
なのにお客様は

「なんかこの店変わったね。」

と仰られました。

その1年後、お店を畳むこととなります。
今思うと、始めた3年目くらいまでは、
「いつきても変わらずいい店だね！」
と言われた事がありました。

周りの飲食店のレベルが上がっている中、お客様も同時にお店を見る目のレベルが上がります。
僕はそれに気づけず、あぐらをかいてしまいました。
その結果、変えていないはずのお店が変わったと言われ、変えていた時期はお客様と一緒に成長していたんだなと身にしみてわかりました。

この経験は今後仕事に取り組む上で僕の土台となる苦い経験ですが、経験して良かったと思える事です。
